# DodleChat

DodleChat is a modern, real-time chat application built with React. It features emoji support, image sharing, speech recognition, and a responsive UI for seamless communication.

## Features

- Real-time messaging
- Emoji picker and reactions
- Image upload and sharing
- Speech-to-text support
- Responsive and modern UI
- Persistent chat history

## Tech Stack

- React
- SWR (data fetching)
- Emoji Mart
- Axios
- React Router
- Electron (for desktop support)
- Firebase (for deployment)

## Getting Started

### Development

```bash
npm install
npm start
```

### Build

```bash
npm run build
```
